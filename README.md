# Customer-Churn-Analysis
Customer Churn Analysis – Exploratory Data Analysis (EDA)

This project focuses on analyzing customer churn data to identify patterns and key factors influencing customer retention. Using Python libraries such as Pandas, Matplotlib, and Seaborn, the project performs data cleaning, visualization, and exploratory analysis to uncover meaningful business insights.

Key Features

Data cleaning and preprocessing

Churn distribution analysis

Customer behavior analysis based on tenure, contracts, and services

Visual insights using count plots, histograms, and pie charts

Actionable findings for reducing customer churn

Tools Used

Python

Pandas, NumPy

Matplotlib, Seaborn

This analysis helps businesses understand churn drivers and supports data-driven retention strategies.
